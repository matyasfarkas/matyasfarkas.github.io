    Fiscal Debt/GDP Trajectory Audit — Big-6 Economies¶

    Date: 2026-06-25 · Branch: feature/trend-cycle-longrun-targets Recursion source of truth: app/src/lib/compute/fiscalAccounting.ts — computeFiscalBlock is the single recursion wired to the displayed debt-to-GDP charts and bands (baseline, scenario, counterfactual, and band paths all go through it). It is a passive accounting recursion: no Bohn primary-balance reaction and no debt-driven risk premium (those live only in the test-only runFiscalRecursion, which has no production caller). There is no separate fiscalProjection.ts file. Law of motion (per quarter, % of GDP): d_t = d_{t-1}·(1+i_q)/(1+g_q) − pb_t/4 + sfa_t/4, with i_q = EFFI/400, g_q = nominal q/q GDP growth (real Δ + deflator Δ), pb = FREV − FPEXP, and the projection using the structural sfaDefault constant (not the noisy forecast SFA column). Snowball = d·(r−g)/(1+g). Debt RISES iff snowball − pb + sfa > 0 (primary deficit, and/or r>g, and/or positive SFA).

    All numbers below were reconstructed from the shipped baseline_levels.json + historical_levels.json + fiscal_meta.json for each bloc, replaying the exact computeFiscalBlock recursion — not assumed. Default view ships with consensus anchors OFF (raw model), so the paths below are what users see today.

    2.3 Diagnosis¶

    The engine keeps US debt flat because the snowball is implausibly favorable. Nominal g averages 7.39%/yr (RW/WN-prior-elevated inflation persistence; recent history is 5.4–5.9%) while EFFI only drifts from ~3.2% to ~4.1% (avg 3.70%), locking r−g = −3.69pp. This single term (−4.4%/yr) almost exactly cancels the deficit + SFA. Secondary error: the WN stationary priors revert revenue UP (~17.6% last-hist → ~19% mean) and primary expenditure to ~21.8%, so pb is only −2.5% — CBO is −3 to −4%.

    This violates the CBO r→g convergence the most of any bloc: CBO has the effective rate climbing toward nominal g (r−g closing to ~0/slightly positive) as low-coupon debt rolls over; the model locks in r−g = −3.7pp. CBO 2025 Long-Term Budget Outlook has gross debt/GDP RISING ~100%→107% over 5y and to ~120%+ over 10y, with a primary deficit widening toward −4%+.

    The current consensus preset in default_anchors.json is also wrong: US.FPEXP_GDP = 17.88 implies a 1.9% primary surplus (US primary spending is ~21–22% of GDP), and FREV_GDP is missing entirely.

    2.4 Recommended anchors (terminal-display δc targets at H=20)¶

    Direction is owned by r−g (driftSetter caveat 3), so a CBO-rising path needs BOTH the snowball and the deficit fixed — widening the deficit alone only lifts debt to ~128 because the −4.4%/yr snowball swamps it.

    Anchor (US) Set to Units Effect
    GDPH (real growth) 1.9 % q/q ann nominal g ↓ to ~3.9–4.8%
    JGDP (deflator) 2.0 % q/q ann (with GDPH)
    EFFI 5.0 % p.a. r−g closes to ~0/+1pp (CBO convergence)
    FPEXP_GDP 21.5 % GDP with FREV → pb = −4.0% (CBO-sized)
    FREV_GDP 17.5 % GDP ADD (currently missing)

    Resulting rising path (verified by recursion): with anchors approached over the horizon via the δc forward-pass (partial application, gMult~0.65 on g), debt/GDP rises monotonically 122.6 → ~125 → 128 → 132 → 137 → ~143 (+~20pp/5y). Drivers then read snowball ~−1 to 0%/yr (neutral), primary deficit −4%/yr, SFA +1.75/yr. Held at full steady-state values from t=0 the recursion reaches ~159; the realized path lands between, ~143. Keep SFA at sfaDefault = 1.752.

    Milder, debt-held-by-public-style slope: EFFI:4.0, pb −3.0% (FPEXP 21.0 / FREV 18.0), g held ~4.5% → +12–15pp over 5y.

    4. How to apply it in the tool¶

    Two mechanisms drive long-run targets, both feeding the δc forward-pass drift setter (driftSetter.ts): - app/public/data/default_anchors.json — the shipped consensus presets per bloc (keyed by variable name, e.g. FREV_GDP, FPEXP_GDP, EFFI, GDPH, JGDP, GDP_<C>, CPI_<C>/HICP_<C>). - The "long-run target" sliders in the UI — the same δc targets, settable interactively (terminal-display value at H=20; lin vars = level, e.g. % of GDP; log vars = % q/q annualized, e.g. growth/inflation/rate).

    To put every big-6 debt/GDP on a realistic increasing trajectory, edit default_anchors.json as follows. (SFA changes are in each bloc's fiscal_meta.json sfaDefault, NOT in default_anchors.json.)

    "US": { "GDPH": 1.9, "JGDP": 2.0, "UI": 2.3, "LR": 4.2,
        "EFFI": 5.0, "FPEXP_GDP": 21.5, "FREV_GDP": 17.5 },   // FIX FPEXP 17.88→21.5, ADD FREV
"EA": { /* + */ "EFFI": 2.9, "GDP_EA": 1.0 /* trims nominal g */ },
"JP": { "GDP_JP": 0.7, "CPI_JP": 2, "LR_JP": 2.5, "YIELD10_JP": 2,
        "FPEXP_GDP": 35.0, "FREV_GDP": 33.5, "EFFI": 2.0 },    // FLIP 33.3/35 → 35/33.5, ADD EFFI
"UK": { "GDP_UK": 1.3, "CPI_UK": 2, "POLRATE_UK": 2.75, "LR_UK": 4.5, "YIELD10_UK": 3.75,
        "EFFI": 4.0, "FREV_GDP": 39.0, "FPEXP_GDP": 41.5 },    // EFFI 3.5→4.0, ADD FREV+FPEXP
"CA": { "GDP_CA": 1.7, "CPI_CA": 2, "LR_CA": 6,
        "EFFI": 4.0, "FREV_GDP": 40.5, "FPEXP_GDP": 41.5 },    // ADD EFFI+FREV+FPEXP
"CN": { "GDP_CN": 4, "POLRATE_CN": 3, "FPEXP_GDP": 32.0, "FREV_GDP": 25.5 }  // ADD FREV, FPEXP 29.1→32.0
    Plus fiscal_meta.json SFA edit: CA sfaDefault: 3.191 → ~0.5. Keep US 1.752, EA 0.325, JP 2.233, UK 3.149 (or trim to ~1.5), CN 4.017.

    Why these levers: the debt direction is governed by r−g plus the primary balance. For US/JP the binding fix is raising EFFI toward g (collapsing the free-lunch snowball) and widening/flipping the deficit; for UK/CA it is adding the missing FREV/FPEXP anchors to kill the spurious WN-prior surplus (the single highest-leverage fix); for EA it is steepening an already-rising path; for CN it is moderating an over-steep but correctly-signed path by anchoring revenue.

    5. Honesty flag — where rising debt would be UNrealistic¶

    • JP (level). The model's d0 ≈ 193% sits on the GG-gross perimeter, below the ~250% headline. The recommended treatment makes the direction realistic (gently rising, IMF Art-IV-consistent: deficit worsening, interest costs doubling by 2030). But do not read the level as the headline 250% — match direction, not level. Forcing a steeper rise than ~+3.7pp/yr would overstate vs IMF.
    • CN (level/perimeter). d0 = 94.3 and pb = −8.9% sit on the augmented-leaning side of the two IMF perimeters (GG ~88%→110% by 2029 vs augmented ~124%→150%+). The current +42pp overshoots even the augmented GG path; the recommendation moderates it to ~120–125 (between GG and full augmented). A larger rise here would be unrealistic — the SFA already encodes the LGFV channel, so do not also widen the deficit.
    • EA aggregate masks dispersion. The +5.4pp rise is the EA-aggregate; it hides IT/FR (~135/115% rising) vs DE (~63% falling). The 4-var block does not model the SGP/debt-brake consolidation channel — treat the EA rise as an unconstrained aggregate, not a rules-consistent forecast.
    • General caveat. "Realistic rising" means matching the benchmark sign and slope (CBO for US, IMF Fiscal Monitor / WEO for the rest), NOT mechanically forcing every bloc up. EA, JP, CN already rise (CN strongly); only US/UK/CA require flipping a flat/falling artifact, and those flips are anchor-driven corrections of over-optimistic WN-prior means, not fabrications.
